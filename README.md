Игра Тетрис.

Сцена содержит изометрическую камеру и TileMap.

Класс CameraScript подключается к камере и управляет игровым процессом. Имеет свойства:
- colorSprites - массив спрайтов для отображения блоков сцены, содержащий 3 элемента: 0- спрайт не занятого блока, 1 и 2 - спрайты блоков, относящихся к фигурам. Цвета блоков фигур закодированы значениями 1 и 2, но могут иметь другие целочисленные значения, для чего массив colorSprites должен быть расширен соответственно и содержать требуемые спрайты.
- wallSprite - спрайт, которым рисуются стены уровня.

Уровни, создаваемые в процессе игры реализованы на базе класса GameLevel. Инициализация уровней производится из класса CameraScript в соответствии с данными из массива levels. Классы уровней по-разному реализуют взаимодействие фигуры со стенками и позволяют генерировать различный набор фигур.

Класс Figure отвечает за отображение фигур. Он обеспечивает расположение фигуры в игровом поле, перемещение фигуры и проверку на столкновения с блоками уровня. 

Объектами типа FigureSkin определяют внешний вид фигуры. В них закодированы блоки фигур для различных улов вращения.
